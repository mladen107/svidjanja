## Description

Povio Labs NodeJS Test Assignment 1.0

## Installation

```bash
$ brew install postgresql
$ createdb -h localhost -p 5432 -U postgres povio-node-development
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
